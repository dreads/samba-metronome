Samba Swing Metronome

===REQUIREMENTS===
-Linux, Unix, or Macintosh based OS
  (Brave Windows users can invoke the metronome by writing a batch file 
  corresponding to the shell scripts, swing_metronome.sh and 
  straight_metronome.sh mentioned below)

-JRE 1.6 or higher.

===HOW TO RUN===

To run the swing metronome, extract the archive and execute swing_metronome.sh
  from a terminal session.
To run the "straight" metronome execute straight_metronome.sh from a terminal
  session

To change the tempo, modify the the com.bowsers.midiutil.tempo value in either 
  swing_metronome.sh or straight_metronome.sh.

===HOW TO MODIFY THE SOUND/SWING===
To modify the instrument, accents, swing feel (more or less swing), or add sauce
  to the swing metronome, modify config/swing_pattern_input.properties

To modify the instrument, accents, swing feel (more or less swing), or add sauce
  to the "straight" metronome, modify config/straight_pattern_input.properties
